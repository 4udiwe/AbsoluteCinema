searchWithFilter

@GET(value = "v1.4/movie")
abstract suspend fun searchWithFilter(@Query(value = "sortField") fields: List<String>?, @Query(value = "sortType") sortTypes: List<Int>?, @Query(value = "type") types: List<String>?, @Query(value = "isSeries") isSeries: Boolean?, @Query(value = "year") years: List<String>?, @Query(value = "rating.kp") kpRating: List<String>?, @Query(value = "genres.name") genres: List<String>?, @Query(value = "countries.name") countries: List<String>?, @Query(value = "lists") inCollection: List<String>?): MoviesResponseDto

Выполняет поиск с различными фильтрами.

  • fields - сортировка по полям из MoviesDTO (year, rating.kp...)

  • sortTypes - тип сортировки для каждого из полей fields ("1", "-1")

  • types - поиск по типам:

    • movie

    • tv-series

    • cartoon

    • animated-series

    • anime

    Пример: {"movie", "tv-series", "!anime"}

  • isSeries - является ли сериалом

  • years - года, как по отдельности, так и диапазоном

    Пример: {1874, 2050, !2020, 2020-2024}

  • kpRating - поиск по рейтингу Кинопоиска

    Пример: { 7, 10, 7.2-10 }

  • genres - жанры

    Пример: { "драма", "комедия", "!мелодрама", "+ужасы" }

  • countries - страны

    Пример: { "США", "Россия", "!Франция" , "+Великобритания" }

  • inCollection - вхождение в какую-либо коллекцию

    Пример: { "top250", "top-100-indian-movies", "!top-100-movies" }
